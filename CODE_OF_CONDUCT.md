# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above.

## Enforcement

Instances of unacceptable behavior may be reported via
[GitHub Security Advisories](https://github.com/yongchoelchoi/tollama/security/advisories/new)
or by contacting the project maintainers directly.

All reports will be reviewed and investigated promptly and fairly.
