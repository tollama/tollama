# Internal Documents

This directory contains maintainer-facing planning documents written in Korean.

- `roadmap.md` — implementation roadmap with status tracking
- `todo-list.md` — detailed task list with priority levels (P0/P1/P2)

These are not required for using or contributing to tollama. See the root
[README.md](../../README.md) and [CONTRIBUTING.md](../../CONTRIBUTING.md) for
contributor-facing documentation.
