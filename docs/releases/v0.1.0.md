## Licensing Decisions

### moirai-2.0-R-small (cc-by-nc-4.0)

**Decision:** Keep in the default model registry with clear non-commercial notice.

The `moirai-2.0-R-small` entry in `model-registry/registry.yaml` uses the
Creative Commons Attribution-NonCommercial 4.0 license. This model is included
in the default registry because:

- The registry entry has `needs_acceptance: true`, requiring explicit user
  consent before pulling weights.
- The entry includes a `notice` field clearly stating:
  _"Non-commercial; research-only weight license. You must accept before pulling."_
- Tollama does not redistribute the model weights; it only references the
  Hugging Face Hub location.

Users are responsible for complying with the model's license terms.

### TimesFM source-archive pinned dependency

**Dependency:** `timesfm[torch,xreg] @ https://github.com/google-research/timesfm/archive/2dcc66fbfe2155adba1af66aa4d564a0ee52f61e.tar.gz`

**License:** Apache-2.0 (verified at pinned commit).

This dependency is only installed when the optional `runner_timesfm` extra is
requested. The pinned commit hash ensures reproducible builds. The license
at the pinned commit is Apache-2.0, which is compatible with tollama's MIT
license.

## Known Limitations

- Version `0.1.0` is a beta release (`Development Status :: 4 - Beta`).
- No training or fine-tuning support (intentional v1 non-goal).
- No distributed scheduler (intentional v1 non-goal).
- VRAM reclaim policy under idle conditions is basic; crash recovery behavior
  for runners is not yet comprehensive.
- Lag-Llama and PatchTST are currently target-only integrations; covariates/static
  features are ignored in `best_effort` mode and rejected in `strict` mode.
- PatchTST quantile outputs depend on backend/runtime capability and may not be
  emitted in every environment.
